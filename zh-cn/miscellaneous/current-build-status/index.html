MariaDB 状态

从 2023 年 2 月 8 日开始，Qt 5.15 及以上的桌面版构建中附带 MariaDB connector/C 的动态库。 这个动态库用来构建 MySQL 数据库插件。
不使用 MySQL 提供的库的原因有以下几个。

  1. MariaDB 是真正的“原来的” MySQL
  2. 使用新版 MySQL（8.0 及以后的版本）需要下载并构建整个 MySQL 数据库。而 MySQL 数据库太大，构建较费事。
  3. MySQL 出自 Oracle。而 Oracle 有打击开源社区的历史。

在 Qt 5.15 版本上，我们使用 MariaDB connector/C 3.1 系列的版本。
在 Qt 6.2 以及更高的版本上，我们使用 MariaDB connector/C 3.3 系列的版本。
